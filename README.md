# College Assignment Repository

## Overview

This repository contains all assignments completed as part of the 4-week college assignment program. The work is organized week-wise and day-wise to maintain a clear learning and submission structure.

### Folder Description

* **OOPS/** → Weekly Object-Oriented Programming assignments and practice work.
* **Day_1 to Day_5/** → Daily assignments, coding exercises, and solutions for the respective week.


## Duration

* Total Duration: **4 Weeks**
* Working Days per Week: **5 Days**

## Objectives

* Strengthen programming and problem-solving skills.
* Practice concepts taught during coursework.
* Maintain consistency through daily assignments.
* Build discipline in coding and documentation.
